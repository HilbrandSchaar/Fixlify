FAQ
Veelgestelde vragen
Hoeveel kan ik echt besparen met de huurcheck?+
Dat hangt af van je woning. Veel huurders betalen €100-€400 per maand teveel ten opzichte van de wettelijke maximumhuur. De Huurcommissie kan dit verschil tot 12 maanden met terugwerkende kracht laten terugbetalen.
Wanneer komen de andere tools online?+
We bouwen elke week 2-3 nieuwe tools. Volg ons op TikTok of Instagram (@fixlifynl) voor updates. De gratis calculators komen eerst, daarna de premium tools en als laatste het Pro abonnement.
Heb ik een account nodig?+
Voor de huidige tools niet. Zodra het Pro abonnement live is heb je daarvoor wel een account nodig — maar voor losse tools blijft het altijd zonder registratie.
Hoe werkt de betaling?+
Bij betaalde tools betaal je via iDEAL, creditcard, Apple Pay of Klarna. Je betaalt eenmalig — geen abonnement (tenzij je voor Pro kiest).
Wat is het verschil tussen losse tools en Pro?+
Losse tools koop je per stuk (€0,50-€9,99 eenmalig). Pro is een maandabonnement (€9,99/m of €99/jaar) waarmee je ALLE betaalde tools onbeperkt kunt gebruiken zonder watermerk, plus toegang krijgt tot premium AI-tools.
Zijn de brieven juridisch geldig?+
Onze brieven zijn voorbeelddocumenten gebaseerd op de juiste wetsartikelen. Dit is geen juridisch advies. Voor specifieke situaties kun je gratis terecht bij het Juridisch Loket.
Hoe zit het met mijn privacy?+
Je gegevens worden lokaal in je browser verwerkt. We gebruiken geen tracking-cookies of analytics. Lees onze privacyverklaring voor meer details.
Kan ik mijn geld terugkrijgen?+
Bij digitale producten vervalt het wettelijke herroepingsrecht zodra de levering begint (art. 6:230p BW). Bij technische problemen kun je natuurlijk altijd contact opnemen via fixlifyNL@outlook.com.
