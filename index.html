FAQ

Veelgestelde vragen

Zijn de tools van Fixlify gratis?+

De meeste tools zijn gratis te gebruiken. Bij sommige tools vragen we een kleine eenmalige bijdrage voor de download (bijvoorbeeld de Huurcommissie Bezwaar brieven). Geen abonnement, geen verrassingen.

Hoe betaal ik voor een tool?+

Betalen gaat veilig via iDEAL of creditcard via Stripe. Je betaalt alleen voor wat je gebruikt — geen abonnement, geen kleine lettertjes.

Kan ik mijn bestanden downloaden?+

Ja. Alle resultaten kun je downloaden als PDF. Direct klaar om te printen, e-mailen of aangetekend te versturen.

Werken de tools ook op mijn telefoon?+

Ja, Fixlify werkt perfect op telefoon, tablet en desktop. Alle tools zijn volledig responsive.

Zijn mijn gegevens veilig?+

We slaan geen persoonlijke gegevens op. Alles wat je invult wordt lokaal in je browser verwerkt. We verkopen nooit data aan derden.

Komen er nog meer tools bij?+

Zeker. We werken continu aan nieuwe tools. De tools met "Binnenkort" staan op de planning. Volg ons op social media om op de hoogte te blijven.

Klaar om te starten?

Gratis proberen. Geen account nodig.
